#Netflix Login Page Clone
This project is a Netflix Login / Get Started page clone created using HTML and CSS. The goal of this project is to practice front-end development skills by recreating the layout and styling of a popular streaming platform’s landing page.

Features:-
1.Responsive layout similar to Netflix’s login/get started page
2.Clean and modern UI design
3.Structured HTML for better readability
4.Styled using pure CSS (no frameworks)


Technologies Used:-

HTML5 – for page structure
CSS3 – for styling and layout


Purpose:-
1.This project is built for learning and practice purposes only. It helps improve understanding of:
2.Page layout and alignment
3.CSS styling techniques
4.Building real-world UI clones
